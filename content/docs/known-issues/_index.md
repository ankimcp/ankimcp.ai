This section documents known issues and limitations when using Anki MCP with AI assistants. Each issue includes a description, workarounds, and current status.

## Current Issues

### [Silent Note Update Failure](viewing-notes-blocks-updates)
Updates appear successful but don't actually save when notes are open in Anki's browser. A documented AnkiConnect limitation with no error message.
